How the real thing would send (all open-source)

This demo only mocks the message. In production, a scheduled job (launchd/cron) re-runs your saved query, diffs against the last run, and sends new items through an open-source notifier — no vendor lock-in:

• Apprise — one Python library, 100+ channels (email + SMS + push). • Novu — self-hostable multi-channel infrastructure. • Listmonk — newsletter / digest manager. • ntfy — lightweight push. Email via Nodemailer/SMTP; SMS via an open gateway (Jasmin / Kannel) or a provider.

What this is

The City Record is NYC's official daily journal: by law (City Charter §1066) every agency publishes its procurement, personnel changes, hearings, land-use actions, and property dispositions there. This tool reads that data live from NYC Open Data and re-links the scattered notices into something a person can follow.

Honest caveats

• Money: RFPs rarely list a value (so “Min award $” applies to Awards); some PINs are junk or blanket codes; raw amounts have outliers (totals exclude rows above $5B).

• People: exam status comes from the Civil Service List, which only lists competitive titles — a title absent there is shown as “no-exam.” Salary band is FY2025 base salary; new hires lag a fiscal year in payroll. Names join across datasets imperfectly (first name is truncated in personnel notices).
